# Rock Paper Scissors

A fun rock-paper-scissors game with multiple modes and stats tracking.

## Features

- 3 game modes: Classic, Best of 5, Best of 10
- Emoji icons (no images needed)
- VS display animation
- Win streak tracking
- Stats bar (wins/losses/draws)
- Keyboard support
- Game over screen
- Responsive design
