# Happy Birthday

An interactive birthday greeting page with confetti, cake animation, and themes.

## Features

- Confetti animation
- Animated cake with candles
- Floating balloons
- 5 color themes
- Age selector
- Sound effects
- Responsive design

## Live Demo

[View Project](https://gemachistesfaye.github.io/SUMMER-WEB/Happy-Birthday/)

## Tech

- HTML5
- CSS3 (Keyframe Animations)
- Vanilla JavaScript
- Canvas API
