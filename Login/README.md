# Login

A premium login page with creative animations and glassmorphism design.

## Features

- Eye-tracking character (pupils follow cursor)
- Floating particles background
- Glassmorphism form
- Password show/hide toggle
- Password strength indicator
- Remember me checkbox
- Social login buttons (Google, GitHub, Twitter)
- Ripple button effect
- Loading spinner on submit
- Mobile-optimized (skips animation on touch)
- Responsive design
