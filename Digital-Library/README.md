# Digital Library

A self-contained digital library with 20 built-in books and no backend required.

## Features

- 20 built-in classic books
- Search by title or author
- Category filters
- Favorite books (LocalStorage)
- Color themes
- Book details modal
- Fully responsive

[View Project](https://gemachistesfaye.github.io/SUMMER-WEB/Digital-Library/)

## Tech

- HTML5
- CSS3 (Grid, Responsive)
- Vanilla JavaScript
- LocalStorage
