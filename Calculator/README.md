# Calculator

A modern calculator with glassmorphism UI and advanced features.

## Features

- Glassmorphism design
- History panel
- Keyboard support
- Ripple click effects
- Percent and negate buttons
- Backspace support
- Responsive design

[View Project](https://gemachistesfaye.github.io/SUMMER-WEB/Calculator/)

## Tech

- HTML5
- CSS3 (Glassmorphism, Backdrop-filter)
- Vanilla JavaScript
