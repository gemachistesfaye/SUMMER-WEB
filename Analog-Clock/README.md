# Analog Clock

A beautiful analog clock with world clocks, stopwatch, and smooth animation.

## Features

- Smooth requestAnimationFrame animation
- Minute and hour ticks
- Glow ring effect
- World clocks (multiple time zones)
- Built-in stopwatch
- Responsive design
