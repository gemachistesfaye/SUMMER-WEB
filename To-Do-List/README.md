# To-Do List

A clean and modern to-do list with priority management and filters.

## Features

- Add, complete, delete tasks
- Priority levels (High, Medium, Low)
- Filter by All / To Do / Done
- Progress bar
- Stats counter
- Smooth animations
- LocalStorage persistence
- Clear completed button
- Responsive design

## Tech

- HTML5
- CSS3 (Clean UI, Responsive)
- Vanilla JavaScript
- LocalStorage
