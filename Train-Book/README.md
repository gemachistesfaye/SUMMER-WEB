# Train Book

A frontend booking system for the Ethiopian-Djibouti Railway.

## Features

- Station selection with swap button
- Date and class selection
- Seat availability display
- Booking confirmation
- Payment summary
- Modern gradient UI
- Responsive design

- Vanilla JavaScript
