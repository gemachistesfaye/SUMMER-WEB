# SUMMER WEB

A collection of **8 interactive front-end projects** built with HTML, CSS & JavaScript during summer break.

## Live Demo

**https://gemachistesfaye.github.io/SUMMER-WEB**

## Projects

| Project | Description | Live Demo |
|---------|-------------|-----------|
| **Analog Clock** | World clocks, stopwatch, smooth animation, glow ring | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Analog-Clock/) |
| **Calculator** | Glassmorphism UI, history panel, keyboard support | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Calculator/) |
| **Digital Library** | 20 built-in books, search, categories, favorites | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Digital-Library/) |
| **Happy Birthday** | Confetti, animated cake, balloons, 5 themes | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Happy-Birthday/) |
| **Login** | Eye-tracking character, particles, glassmorphism | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Login/) |
| **Rock Paper Scissors** | 3 game modes, emoji icons, streak tracking | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Rock-Paper-Scissors/) |
| **To-Do List** | Priority levels, filters, progress bar | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/To-Do-List/) |
| **Train Book** | Ethiopian railway booking system frontend | [Live](https://gemachistesfaye.github.io/SUMMER-WEB/Train-Book/) |

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, Animations, Responsive)
- Vanilla JavaScript
- Font Awesome Icons
