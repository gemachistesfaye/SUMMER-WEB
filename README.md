# SUMMER WEB

A collection of **8 interactive front-end projects** built with HTML, CSS & JavaScript during summer break.

## Live Demo

**https://gemachistesfaye.github.io/SUMMER-WEB**

| Project | Description | Live |
|---------|-------------|------|
| **Analog Clock** | World clocks, stopwatch, smooth animation, glow ring | [View](Analog-Clock/) |
| **Calculator** | Glassmorphism UI, history panel, keyboard support | [View](Calculator/) |
| **Digital Library** | 20 built-in books, search, categories, favorites | [View](Digital-Library/) |
| **Happy Birthday** | Confetti, animated cake, balloons, 5 themes | [View](Happy-Birthday/) |

| **Train Book** | Ethiopian railway booking system frontend | [View](Train-Book/) |

- HTML5
- CSS3 (Glassmorphism, Animations, Responsive)
- Vanilla JavaScript
- Font Awesome Icons
